## 🚀 Featured Projects

> Recent research & coursework from the UT Austin MS AI program.

### ⚡ LLM Memory Optimization — LoRA, QLoRA & 4-bit Quantization
Implemented four memory-efficient variants of a 73MB network from scratch (half-precision, LoRA adapters, 4-bit block quantization, QLoRA), achieving up to **7× memory reduction (72MB → 11MB)** with matched forward-pass outputs. Both LoRA and QLoRA converged on a 1000-sample classification task within ~30 iterations.
`PyTorch` · `LoRA` · `QLoRA` · `INT4 Quantization`

### 🎯 LLM Fine-Tuning — SFT, Chain-of-Thought & Rejection Sampling (RFT)
Fine-tuned SmolLM2-1.7B for unit conversion using SFT with LoRA adapters; built batched generation with left-padding for **10–20× inference speedup**. Built an RFT data-generation pipeline (CoT sampling, correctness filtering) that outperformed vanilla SFT with **>90% generation success**.
`SmolLM2` · `HuggingFace` · `PEFT` · `LoRA` · `RFT`

### 🤖 LLM-Guided Robotic Manipulation
Closed-loop framework combining LLM symbolic reasoning with imitation-learned motor skills (ResNet-18 visual tokens + Behavior Cloning on Robomimic). Achieved **92% / 88% / 64%** success on lift/place/stack vs. 18% / 16% / 4% for LLM-only; ReAct-style prompting cut decisions/episode from 27.5 to 8.2.
`PyTorch` · `PyBullet` · `ResNet-18` · `Behavior Cloning` · `LLM`

### 🖼 Autoregressive Image Generation with BSQ Tokenizer
Trained a patch-level autoencoder + Binary Spherical Quantizer (10-bit codebook) for **~7× compression (500MB → 76MB)**, then a decoder-only Transformer with causal masking for next-token image generation, reaching <4500 bits/image cross-entropy.
`PyTorch` · `Transformers` · `Binary Spherical Quantization`

### 👁 Computer Vision & NLP — CNN / U-Net / Transformer + ELECTRA / DAN
U-Net road detector (segmentation mIoU > 0.75, depth MAE < 0.05) and a Perceiver Transformer planner for autonomous driving. ELECTRA-small fine-tuned on SNLI (~89% accuracy) with contrast-set debiasing; typo-robust DAN maintained >74% accuracy on corrupted data with zero retraining.
`PyTorch` · `HuggingFace` · `GloVe`
